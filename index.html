Version Chart

This is a version visualization made by Felix Nagel. More info

This Java-Script application is powered by jQuery and jQuery UI.
The used data (link to json file) is provided by the TYPO3 server team and queried via YQL. Thanks for providing such great tools and services!

Please report bugs via GitHub's Bugtracker (Issues) or make a patch and send an pull request.
Any additional information regarding specific TYPO3 version will be added, so please feel free to add feature requests!
